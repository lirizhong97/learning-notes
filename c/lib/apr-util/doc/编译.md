# Apache Portable Runtime Util library

```
http://apr.apache.org/download.cgi
```

```
wget https://dlcdn.apache.org//apr/apr-util-1.6.1.tar.gz
```

```
tar -zxvf apr-util-1.6.1.tar.gz
```

```
cd apr-util-1.6.1
```

```
./configure --with-apr=/usr/local/apr
```

```
make
```

```
sudo make install
```
